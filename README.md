# Marketplace API

API для управления маркетплейсом: пользователи, продукты, заказы, корзина и статистика.

## 🔹 Стек технологий

* **Backend:** PHP 8.3, Laravel 11
* **Базы данных: PostgreSQL 15, Redis
* **Аутентификация:** JWT (JSON Web Token)
* **Тестирование:** PHPUnit
* **Дополнительно:** Docker (контейнеризация)

## 🔹 Структура проекта

```
app/
├─ Console/Commands            # CLI команды
├─ Contracts/                  # Контракты
├─ DTO                         # Data Transfer Objects для передачи данных между слоями
├─ Exceptions                  # Исключения
├─ Http/
│  ├─ Controllers/Api          # Контроллеры API
│  ├─ Middleware               # Middleware
│  └─ Requests                 # Валидация запросов
├─ Models                      # Eloquent модели
├─ Services                    # Бизнес-логика
└─ Traits                      # Трейты
```

## 🔹 Основные модули API

1. **Auth** – регистрация, логин, токены.
2. **Product** – CRUD продуктов, хранение информации о товарах.
3. **Cart** – добавление/удаление продуктов из корзины.
4. **Order** – создание и просмотр заказов.
5. **Admin** – управление пользователями, продуктами, статистикой.
